# Vehicle Diagnostic Interface using Verilog HDL

## Overview
This project implements a simplified FPGA-based Vehicle Diagnostic Interface using Verilog HDL. The system receives Diagnostic Trouble Codes (DTCs) through UART communication, decodes the received fault, stores the fault information, and indicates the severity using Green, Yellow, and Red LEDs.

## Features
- UART-based serial communication
- Baud Rate Generator
- UART Receiver
- Frame Decoder
- Diagnostic Decoder
- Fault Register
- LED Controller
- Modular Verilog HDL design
- Simulated using Xilinx ISE 14.7 (ISim)
